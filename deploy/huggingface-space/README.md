# Continuum

AI for humans by VANTA Research.

## About

Continuum is a conversational AI interface that supports multiple model providers including OpenAI, Anthropic, Mistral, OpenRouter, and local models via llama.cpp.

## Getting Started

1. Navigate to the Settings page
2. Configure your preferred AI provider
3. Add your API key (if using a cloud provider)
4. Start chatting!

## Features

- Multi-provider support (OpenAI, Anthropic, Mistral, OpenRouter, Local)
- Model downloads from HuggingFace
- Conversation memory
- Project organization
- Beautiful, responsive UI
